\# Eclipse 多功能工具箱



东方Project角色与装备数据管理工具，提供角色总面板计算、角色数据库和装备数据库功能。



\## 功能特点



\- \*\*角色总面板计算\*\*：计算角色+装备+继承的总面板属性

\- \*\*角色数据库\*\*：浏览和筛选所有角色及其属性

\- \*\*装备数据库\*\*：浏览和筛选所有装备及其属性

\- \*\*响应式设计\*\*：支持桌面和移动设备

\- \*\*离线使用\*\*：支持PWA，可安装为本地应用



\## 使用方法



1\. 访问 \[GitHub Pages 网站](https://your-username.github.io/eclipse-toolbox/)

2\. 选择所需功能：

&nbsp;  - 角色总面板计算：计算角色装备搭配

&nbsp;  - 角色数据库：浏览角色数据

&nbsp;  - 装备数据库：浏览装备数据



\## 技术栈



\- 纯HTML/CSS/JavaScript

\- 响应式设计

\- PWA（渐进式Web应用）

\- GitHub Pages部署



\## 本地开发
